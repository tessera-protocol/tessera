# Tessera Mark Variants

- `tessera-mark-purple.svg`: Primary single-color mark for standard brand applications on light backgrounds.
- `tessera-mark-twotone.svg`: Two-tone mark for marketing, product, and presentation contexts where the brand palette should be visible.
- `tessera-mark-mono.svg`: Neutral dark mark for monochrome layouts, print, and low-color contexts.
- `tessera-mark-reversed.svg`: White mark for dark, saturated, or photographic backgrounds.
